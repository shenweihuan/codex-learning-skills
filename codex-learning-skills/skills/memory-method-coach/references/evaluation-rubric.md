# Evaluation Rubric

Use this rubric whenever improving or judging a memory-method output. Score each dimension from 1-5.

## Dimensions

1. **Exact recall**: The cue leads back to the exact word, phrase, order, definition, or decision rule.
2. **Image strength**: The mnemonic is concrete, visual, active, and distinctive enough to survive forgetting.
3. **Exam fit**: The output supports timed exam use instead of only being entertaining.
4. **Cognitive load**: The method is short enough for the user's time and mental energy.
5. **Closed loop**: The output includes active recall, error traps, and a review point.

## Score Meaning

- 5: Ready to use. The user can test it immediately without extra explanation.
- 4: Good, with one small weakness.
- 3: Usable but likely to decay or confuse under exam pressure.
- 2: Interesting but not reliable.
- 1: Does not help recall the target.

## Pass Gate

An output passes only if:

- Total score is at least 20/25.
- Exact recall is at least 4.
- Exam fit is at least 4 for exam tasks.
- Closed loop is at least 4.

If it fails, revise the mnemonic before giving the final answer.

## Before/After Optimization Notes

Baseline weaknesses found in dry-run tests:

- Vocabulary outputs often gave a vivid association but did not force spelling or example-sentence recall.
- Writing-template outputs used memory palace but did not always require paraphrase, making dead memorization too likely.
- Abstract-knowledge outputs explained the concept but did not always separate understanding from recall cue.

Required improvement:

- Add one self-test to every output.
- Add one common trap or confusable item.
- Add one short review schedule.
- For learning-mode users, preserve feedback scoring instead of only giving a finished mnemonic.
