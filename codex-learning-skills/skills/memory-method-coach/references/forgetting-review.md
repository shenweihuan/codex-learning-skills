# Forgetting-Aware Review

## Principle

Forgetting is not a failure signal. It is the moment to strengthen retrieval. Use short, spaced recall sessions instead of long rereading sessions.

## Default Review Ladder

For ordinary study material:

1. Immediate recall: 3-10 minutes after learning.
2. Same-day recall: before sleep or after class.
3. Next-day recall: without looking first.
4. Three-day recall: mix with similar material.
5. Pre-exam recall: one-page active recall, not full rereading.

For urgent exams within 7 days:

1. Learn and test now.
2. Retest after 20-40 minutes.
3. Retest before sleep.
4. Retest next morning.
5. Retest the day before exam.

## Repair Repeated Forgetting

If the user forgets the same item repeatedly:

1. Check whether the cue points to the exact target.
2. Make the image more concrete and unusual.
3. Add a contrast against the most common wrong answer.
4. Reduce the item to a smaller chunk.
5. Require one production test: write, speak, or solve from memory.

## Review Output Format

[recall task]

## 明天

## 三天后

## 考前
[one-page final recall]
