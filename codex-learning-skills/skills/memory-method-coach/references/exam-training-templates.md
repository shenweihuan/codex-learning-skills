# Exam Training Templates

## English Vocabulary

Use image association plus example sentence recall.

Output:
- word
- sound/shape hook
- vivid image
- Chinese meaning
- spelling cue
- one exam sentence
- one blank recall prompt
- one confusable trap if relevant

CET-4 sample pattern:

```markdown
| Word | Meaning | Hook | Spelling Cue | Exam Blank |
|---|---|---|---|---|
| abandon | 放弃 | a band on stage suddenly leaves | a-ban-don has 3 beats | Never ___ your plan too early. |
```

After the table, ask the learner to cover the meanings and recover: spelling -> meaning -> sentence.

## CET-4 Writing

Use memory palace for essay frames.

Locations:
1. Door: opening topic sentence.
2. Desk: reason 1.
3. Window: reason 2.
4. Clock: example or consequence.
5. Exit: conclusion and call to action.

Require the user to rewrite the template in their own words.

For CET-4 writing, prefer "skeleton memory" over paragraph memorization:

1. Recite the five-part structure.
2. Write the skeleton from memory.
3. Imitate it with a new topic.
4. Replace at least three expressions so it does not become rigid dead memorization.

Quality gate:

- Can the user recall the order in 30 seconds?
- Can the user produce a new topic version?
- Does the template avoid unnecessarily difficult wording?

## Translation Themes

Use chunking by theme: culture, technology, education, environment, economy, campus life.

For each theme:
- 5 core nouns
- 5 verbs
- 3 sentence patterns
- one active recall mini-test

## Concepts and Definitions

Use contrast table plus image association.

Fields:
- term
- plain definition
- trigger condition
- opposite/confusable term
- visual cue
- recall question

For abstract learning concepts, separate understanding from memory:

- Understanding: one plain-language causal relationship.
- Memory cue: one image or route that retrieves the relationship.
- Test: explain without looking, then answer one application question.

## Formulas

Use story or palace only for structure. Require problem-solving practice for true mastery.

Fields:
- formula
- what each symbol means
- when to use
- common trap
- one sample problem
