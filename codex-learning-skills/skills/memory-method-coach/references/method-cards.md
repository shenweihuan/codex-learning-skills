# Method Cards

## Memory Palace

Best for ordered material: essay structure, speech outlines, process steps, historical sequence, formula derivation.

How to use:
1. Choose a familiar route with 5-12 fixed locations.
2. Put one knowledge point at each location.
3. Make the image exaggerated and physically interacting with the place.
4. Walk the route from memory and say the target knowledge aloud.

Common mistakes:
- Vague images such as "knowledge" or "importance".
- Too many items at one location.
- Forgetting to test without looking.

## Image Association

Best for vocabulary, terms, names, definitions, and abstract concepts.

How to use:
1. Extract a sound, shape, keyword, or meaning hook.
2. Link it to a concrete image.
3. Make the image move, collide, smell, shout, or transform.
4. Add the exact target answer at the end.

Common mistakes:
- Funny image but no retrieval path to the answer.
- Association only reminds the user of the topic, not the exact word.

## Story Chain

Best for unordered lists that must be recalled as a group.

How to use:
1. Turn every item into an object or actor.
2. Put them into a single strange but causal story.
3. Keep the story short enough to retell in under 30 seconds.
4. Test by retelling the story and extracting the list.

Common mistakes:
- Story is entertaining but changes the order accidentally.
- Too many decorative details.

## Peg System

Best for numbered lists and fixed exam outlines.

How to use:
1. Use stable pegs: 1 candle, 2 swan, 3 ear, 4 flag, 5 hook, etc.
2. Attach each target item to the matching peg.
3. Test by number first, then by target first.

Common mistakes:
- Changing the peg list every time.
- Using pegs when the order does not matter.

## Chunking

Best for large, dense material.

How to use:
1. Group items by function, topic, timeline, cause, or contrast.
2. Name each group with a short label.
3. Memorize the labels first, then the items inside.
4. Use a blank-page recall test.

Common mistakes:
- Grouping by page order instead of meaning.
- Creating too many groups.

## Contrast Table

Best for similar concepts, grammar choices, theories, and exam distractors.

How to use:
1. Choose 2-4 easily confused items.
2. Compare by definition, trigger condition, example, and common trap.
3. Convert each row into one retrieval question.

Common mistakes:
- Writing a table but never testing it.
- Comparing superficial wording instead of decision rules.
