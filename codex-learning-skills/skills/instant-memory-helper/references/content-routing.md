# Content Routing

## Word List

Use sound/shape hook and one sentence. For more than 20 words, chunk by topic or prefix/suffix.

## Definitions

Use plain definition, image hook, contrast, and recall question.

## Ordered Process

Use memory palace. The route must have the same number of locations as steps.

## Comparison

Use a decision table. The goal is choosing correctly under exam pressure.

## Formula

Use meaning cue plus problem practice. Do not let mnemonic replace calculation.

## Essay Or Speech

Use a palace or skeleton. Memorize the structure, not a rigid paragraph, unless the exam requires exact wording.

## Mixed Notes

First extract:
1. must-know concepts
2. easily confused pairs
3. ordered procedures
4. likely exam prompts

Then build separate small memory packages.
