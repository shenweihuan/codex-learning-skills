# Urgent Review Mini-Schedules

## Same-Day Exam

1. Learn the memory package now.
2. Recall after 10 minutes.
3. Recall after 30-40 minutes.
4. Do a final blank recall before entering the exam.

## Exam Tomorrow

1. Learn and test now.
2. Test before sleep.
3. Test after waking.
4. Test 1-2 hours before exam.

## Exam This Week

1. Learn today.
2. Recall tomorrow.
3. Mix with similar items in 2-3 days.
4. Final one-page recall the day before exam.

## Minimum Viable Review

If the user is exhausted, use one cycle:

1. Look once.
2. Cover it.
3. Say it.
4. Check it.
5. Fix only what failed.
