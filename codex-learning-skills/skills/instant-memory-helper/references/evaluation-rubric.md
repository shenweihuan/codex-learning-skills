# Evaluation Rubric

Use this rubric before finalizing an urgent memory package. Score each dimension from 1-5.

## Dimensions

1. **Exact recall**: The mnemonic retrieves the exact target, not just a vague topic.
2. **Image strength**: The cue is concrete, active, and easy to picture.
3. **Exam fit**: The package helps in timed test conditions.
4. **Cognitive load**: The package is short enough for a tired or rushed user.
5. **Closed loop**: The output includes a 30-second test, traps, and a review plan.

## Pass Gate

An urgent output passes only if:

- Total score is at least 20/25.
- Cognitive load is at least 4.
- Exact recall is at least 4.
- Closed loop is at least 4.

If it fails, simplify first. Do not add theory to compensate.

## Required Fixes From Dry Run

- Vocabulary packages need spelling and sentence recall, not only meaning hooks.
- Writing packages need a skeleton plus imitation task, not a memorized paragraph.
- Abstract concepts need a one-line decision rule and a concrete image.
- Every output needs a 30-second test.
