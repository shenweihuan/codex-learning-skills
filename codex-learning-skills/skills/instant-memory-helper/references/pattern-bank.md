# Pattern Bank

## List To Story

Use for 4-12 items.

Format:

```markdown
顺序: [items]
故事: [short causal story]
提取: [item1 -> cue, item2 -> cue]
自测: 合上内容，把故事讲一遍，再写出原列表。
```

## CET-4 Word Quick Pack

Use when the user needs fast vocabulary memory.

```markdown
| Word | Meaning | Quick Hook | Spell/Use Check |
|---|---|---|---|
| [word] | [meaning] | [concrete image] | [blank sentence or spelling cue] |
```

Keep each hook short. If words are confusable, use the comparison pattern instead of separate hooks.

## Abstract Term To Image

Format:

```markdown
术语: [term]
核心意思: [plain meaning]
图像钩子: [image]
别混淆: [confusable item]
自测: 看到 [cue] 时说出定义和区别。
```

## Comparison To Decision Rule

Format:

```markdown
如果题目强调 [condition A] -> 选/用 [concept A]
如果题目强调 [condition B] -> 选/用 [concept B]
一句话区别: [contrast]
陷阱: [common wrong choice]
```

For CET-4 confusable words, always add one mixed blank quiz.

## Essay Template To Palace

Use 5 locations:

1. Door: topic
2. Table: reason
3. Window: example
4. Clock: consequence
5. Exit: conclusion

Ask the user to walk the route and speak the essay frame in their own words.

For urgent CET-4 writing:

```markdown
5点骨架: 现象 -> 原因 -> 例子 -> 建议 -> 结论
两个替换主题: [topic A] / [topic B]
30秒自测: 不看内容说出5点，并把其中2点替换成明天可能考的主题。
```

## Formula To Meaning Cue

Format:

```markdown
公式: [formula]
每个符号: [symbol meaning]
使用条件: [when to use]
记忆句: [short cue]
必须练: [one quick problem]
```

## Abstract Concept Pack

Use one-line meaning plus image. Avoid long lectures.

```markdown
概念: [term]
一句话: [plain meaning]
图像: [cue]
应用题: [one quick when-to-use question]
```
