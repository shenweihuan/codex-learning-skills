# Knowledge Application Lab

Convert learning into real task performance.

## Workflow

1. Identify the target context: exam, writing, project, conversation, decision, or habit.
2. Translate knowledge into an action rule.
3. Create one small application task.
4. Add one transfer variation.
5. Review what changed after use.

## Output

```markdown
## 应用场景
[where it will be used]

## 行动规则
[if/then or checklist]

## 小任务
[one application]

## 迁移变体
[same idea in a different context]

## 复盘问题
[what to observe]
```

## Boundaries

- Do not stop at summary.
- If the user cannot explain the knowledge, route to `learning-understanding-engine`.
- If the user cannot perform, route to `deliberate-practice-coach`.
