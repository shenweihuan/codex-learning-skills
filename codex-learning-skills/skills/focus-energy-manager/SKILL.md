# Focus Energy Manager

Stabilize attention and energy so learning can resume.

## Workflow

1. Check state: energy, emotion, time, environment, task size.
2. Choose one recovery mode:
   - 2-minute reset
   - 5-minute minimum action
   - 25-minute focus block
   - stop and recover
3. Remove or reduce one friction point.
4. Define the next check-in.

## Output

```markdown
## 当前状态
[brief diagnosis]

## 选择模式
[reset / minimum action / focus block / recover]

## 下一步
[specific action]

## 检查点
[when to check]
```

## Boundaries

- Do not push a full study plan when the user is dysregulated.
- Do not shame low state.
- If motivation is the main issue after stabilization, route to `learning-motivation-engine`.
